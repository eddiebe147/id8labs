## ğŸ¯ Strategic Placement

### Why These Locations?

**Hero Section** (Multi-layer)
- First impression
- Maximum visual impact
- Sets tropical lab tone
- Brand personality showcase

**Featured Product** (Circuit-organic)
- Premium feel for shipping product
- Tech-nature hybrid reinforces innovation
- Differentiates from other products
- Subtle elevation

**Footer** (Minimal dots)
- Completes visual journey
- Subtle callback to hero
- Professional finish
- Not overwhelming

**What's NOT Textured**
- Header (clean navigation)
- Mission section (focus on message)
- Product grid cards (clean product info)
- Content areas (readability priority)

## ğŸ› ï¸ Customization Guide

### Increase Texture Visibility
```tsx
// Hero - make more prominent
<div className="absolute inset-0 texture-layered-tropical opacity-80" />
<div className="absolute inset-0 texture-mycelium opacity-90" />

// Footer - make stronger
<div className="absolute inset-0 texture-tropical-dots opacity-75" />
```

### Decrease Texture Visibility
```tsx
// Hero - make more subtle
<div className="absolute inset-0 texture-layered-tropical opacity-30" />
<div className="absolute inset-0 texture-mycelium opacity-40" />

// Footer - make lighter
<div className="absolute inset-0 texture-tropical-dots opacity-25" />
```

### Change Patterns
```tsx
// Replace hero texture
<div className="absolute inset-0 texture-circuit-organic" />

// Replace footer texture
<div className="absolute inset-0 texture-dither-noise opacity-40" />
```

### Add to New Sections
```tsx
// Mission section example
<section className="relative">
  <div className="absolute inset-0 texture-mycelium opacity-30" />

    {/* Content */}
  </div>
```
