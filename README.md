# ID8Labs

**White Paper Edition**
Clean, scientific website for ID8Labs - Professional tools for the AI era.

## Design Philosophy

This is the lab coat. Boring. Scientific. Professional. Credible.
Products bring the personality. The lab just needs to be trusted.

## Tech Stack

- **Next.js 14** - App Router
- **TypeScript** - Strict mode
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Subtle animations
- **Radix Icons** - Minimal iconography

## Design System

### Monochromatic Only

**Light Mode:**
- Background: `#FFFFFF` (pure white)
- Text: `#000000` (black) / `#666666` (gray)
- Borders: `#E5E5E5`

**Dark Mode:**
- Background: `#0A0A0A` (near black)
- Text: `#FFFFFF` / `#999999`
- Borders: `#2A2A2A`

### Typography

- Font: Inter
- H1: 3.5rem (700 weight)
- H2: 2.5rem (700 weight)
- Body: 1.125rem (400 weight)

### Layout

- Max width: 1200px
- Section spacing: 6rem (96px)
- Generous whitespace everywhere

## Getting Started
